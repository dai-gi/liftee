# liftee

## 開発言語

- Ruby 3.2.2
- Rails 6.1.7.2
- Vue 3.3.4
- Node 18.16.0

</br>

## 実行手順

```plain text
$ git clone git@github.com:dai-gi/liftee.git
$ cd liftee
$ bundle install
$ rails db:create && db:migrate
$ rails s
```
